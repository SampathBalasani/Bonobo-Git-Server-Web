Bonobo Git Server

Simple git server for Windows.

Set up your own self hosted git server on IIS for Windows. Manage users and have full control over your repositories with a nice user friendly graphical interface.

Download 6.1.0

See the changelog

Simple

The installation is very easy and you can manage everything from a nice and clean graphical interface.

You can use your current Windows infrastructure and Bonobo Git Server runs on your IIS.
